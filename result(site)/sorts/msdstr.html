MSD для строк разной длины

Код находится в пакете ru.mail.polis.sort в файле MSDString.java.

Тесты находятся в пакете ru.mail.polis.sort.valid в файле MSDStringTest.java и называются:
  • test01_checkMSDStringSort() - для массива из одной строки.
  • test02_checkMSDStringSort() - для массива из 10 строк.
  • test03_checkMSDStringSort() - для массива из 100 строк.
  • test04_checkMSDStringSort() - для массива из 1000 строк.
  • test05_checkMSDStringSort() - для массива из 10000 строк.

Бенчмарк находится в пакете ru.mail.polis.bench в файле MSDStringBench.java.
